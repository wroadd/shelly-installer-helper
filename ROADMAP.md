# Roadmap

## Planned

- Optional Gen2+ RPC URL templates alongside legacy-style HTTP endpoint URLs.
- Import discovered devices from a local discovery export when available.
- Installer training mode with extra wiring checks and guided examples.
- More validation warnings for mode-specific channel rules.

## Done

- Multi-language interface support.
  - English remains the default language.
  - Language selector added to the extension options page.
  - Popup, options page, floating panel, status messages and helper templates use locale strings.
  - Added Hungarian, Czech, Slovenian, Slovak, Romanian, French, Spanish, German, Dutch, Italian, Serbian, Croatian, Polish and Portuguese translations.

## Translation Follow-Up

- Native-speaker review for all non-English translations.
- Add automated checks for hard-coded user-facing strings outside `locales.js`.
