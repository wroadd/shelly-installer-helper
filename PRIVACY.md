# Privacy Policy

Shelly Installer Helper is designed to work locally in the browser on Shelly
device web interfaces.

## Data Collection

The extension does not collect, sell, share or transmit personal data.

## Network Access

The extension may send short API requests to the currently open HTTP host to
detect whether the page belongs to a Shelly device. Examples:

- `/rpc/Shelly.GetDeviceInfo`
- `/rpc/Shelly.GetStatus`
- `/rpc/Shelly.GetConfig`
- `/shelly`
- `/status`
- `/settings`

These requests are sent only to the device or host already opened in the active
browser tab.

## Local Storage

The extension stores settings locally in Chrome storage, including:

- manually allowed hosts,
- manually denied hosts,
- default helper settings.

This data stays in the user's browser.

## Clipboard

The extension uses clipboard permission only after a user clicks a copy button.
It copies generated URLs or text templates chosen by the user.

## Remote Code

The extension does not load remote scripts, fonts, styles or executable code.

Use the GitHub repository issues page for questions or reports.
